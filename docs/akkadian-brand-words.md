# Akkadian Brand Words

A word list for naming things — services, features, routes, products. Qabû
started the tradition (qabû = "to say, speak"), `/bab` continued it (bābu =
"gate"). This is the quarry for the next one.

**Pronunciation crib:** `š` = sh · `ḫ` = ch as in Bach (ח) · `ṣ ṭ q` = emphatic
s/t/k (like Hebrew צ ט ק originally) · circumflex (`û â î ê`) = long vowel.
Roughly: qabû = "ka-BOO", bābu = "BAH-boo". Transliterations follow standard
Assyriological convention, forms are mostly Old Babylonian. Akkadian is
Semitic, so Hebrew cognates (≈) abound — they make the words easy to remember.

Already in use: **qabû** (the brand), **bābu** (the `/bab` authenticated route).

## Speech & communication (core brand territory)

| Word | Meaning | Notes |
|---|---|---|
| **qabû** | to say, speak, command | The brand. ✓ |
| **nabû** | to name, call, proclaim | Also the god Nabû, divine scribe (see Gods). ≈ נביא. Rhymes with qabû — strong sibling name. |
| **zikru** | utterance, name, mention | ≈ זכר (remember/mention). |
| **awātum / amātu** | word, matter, affair | "The word" — weighty, legal flavor. |
| **dabābu** | to speak, talk | ≈ דבב (דובב שפתי ישנים). Also "legal case." |
| **pû** | mouth; speech, testimony | ≈ פה. Two letters — great for a tiny service. |
| **lišānu** | tongue, language | ≈ לשון. Nice for i18n / EN-HE anything. |
| **šemû** | to hear, listen | ≈ שמע. The listening half of a chat agent. |
| **apālu** | to answer, respond | What Qabû does all day. |
| **šapāru** | to send, dispatch, write | Root of everything outbound. |
| **šipirtu** | message, letter, instruction | Good for DMs / messaging features. |
| **mār šipri** | messenger, envoy (lit. "son of the message") | A webhook forwarder's true name. |
| **nāgiru** | herald, town crier | Perfect fit for the notifier service. |
| **annu** | yes, consent, approval | "annu kīnu" = a firm yes. Cute for confirmations. |
| **karābu** | to greet, bless | ≈ (maybe) כרוב. Good for the greeting feature. |
| **šulmu** | well-being, peace; a greeting | ≈ שלום. Health checks, status pages. |

## Gates, guarding & trust (infra / security)

| Word | Meaning | Notes |
|---|---|---|
| **bābu** | gate, door | The `/bab` route. ✓ Bāb-ilim = Babylon, "gate of the god." Same word as Aramaic בבא — Talmud's Bava Kamma/Metzia/Batra are literally "first/middle/last gate." |
| **abullu** | city gate | The big public gate where court and commerce happened. |
| **daltu** | door | ≈ דלת. |
| **petû** | to open | "bāba petû" — to open the gate. Login/unlock flavor. |
| **edēlu** | to bolt, lock | The opposite. |
| **erēbu** | to enter | ≈ ערב (בוא השמש). "bāba erēbu" — enter the gate. Onboarding. |
| **atû** | doorkeeper, gatekeeper | The gatekeeper LLM already exists — this is its name. |
| **maṣṣartu** | watch, guard duty; **maṣṣāru** — guard | ≈ משמרת/שומר root (נצר). Monitoring, health checks. |
| **dūru** | city wall, fortress | A firewall, literally. ≈ (Aram.) דור? Prestigious in city names (Dūr-Šarrukīn). |
| **kunukku** | cylinder seal; sealed document | Authentication token, signature — the Bronze Age JWT. |
| **kanāku** | to seal | The signing action. |
| **kittu** | truth, justice, reliability | ≈ כן/כנות root. "kittu u mīšaru" — truth and justice. |
| **kīnu** | true, firm, reliable | ≈ כן. Nice adjective for anything trustworthy. |
| **riksu / rikistu** | bond, contract, treaty | ≈ (loose) קשר concept. Agreements, subscriptions. |
| **markasu** | rope, bond; center, hub | Babylon was "markas šamê u erṣeti" — the bond of heaven and earth. A router/hub name. |

## Quick service-name shortlist

Pre-matched to things that exist or are planned:

- gatekeeper LLM → **atû** (doorkeeper) or **maṣṣāru** (guard)
- notifier → **nāgiru** (herald)
- dashboard/analytics → **nikkassu** (accounts) or **napḫaru** (total)
- backup → **miḫru** (copy, counterpart)
- messaging/DM features → **šipirtu** (message), **mār šipri** (messenger)
- auth/tokens → **kunukku** (seal)
- onboarding → **erēbu** (to enter) / **bāba petû** (open the gate)
- monitoring → **maṣṣartu** (the watch)
- KB → **naqbu** (the Deep) or **apsû**

*Caveat: compiled from Claude's knowledge of Akkadian (standard Assyriological
sources), not checked against a dictionary. Confidence is high for meanings,
lower for exact vowel lengths. Before a word goes on anything public-facing,
verify in the CAD (Chicago Assyrian Dictionary, free online) or ePSD/Oracc.*
