Chat Widget Demo

1. Embedded Mode

Widget embedded in a container with custom configuration:

Usage:

<div id="chat-container" style="height: 600px; width: 400px;"></div>

<script>
  window.ChatWidgetConfig = {
    targetElement: '#chat-container',
    apiUrl: '/custom-api/ask',
    initialMessage: 'Hello! How can I help you today?',
    placeholder: 'Ask me anything...'
  };
</script>
<script src="widget.js"></script>

2. Default Mode (Slide-in)

Click the chat button in the bottom-right corner to open the default sliding widget.

This mode doesn't require a target element and will slide in from the side.

Usage:

<!-- No configuration needed for default mode -->
<script src="widget.js"></script>

<!-- Or with custom config -->
<script>
  window.ChatWidgetConfig = {
    apiUrl: '/api/chat',
    initialMessage: 'Hi there! ðŸ‘‹',
    placeholder: 'Type here...'
  };
</script>
<script src="widget.js"></script>

Configuration Options

{
  targetElement: '#container',     // String selector or DOM element
                                    // If provided: embedded mode
                                    // If null/undefined: default slide-in mode

  apiUrl: '/custom/api/endpoint',  // Custom API endpoint
                                    // Default: window.location.origin + '/widget-api/ask'

  initialMessage: 'Welcome!',      // Custom initial bot message
                                    // Default: 'Need help?' (or RTL equivalent)

  placeholder: 'Type here...'      // Custom input placeholder
                                    // Default: 'Type your message...' (or RTL equivalent)
}
